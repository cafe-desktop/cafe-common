NAME
  cafe-autogen - generates all makefiles for CAFE packages
SYNOPSIS
  cafe-autogen
DESCRIPTION
  cafe-autogen generates all makefiles for CAFE packages
  
  To make use of the common autogen script, create a stub autogen.sh
  script in your module. Take the example in
  autogen.sh.example file in cafe-common documentation folder
SEE ALSO
  autoconf(1), automake(1), autoreconf(1), cafe-doc-common(1)
